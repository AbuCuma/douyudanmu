### 使用方式
1. 打开房间，切换到h5播放器，等右边弹幕区域加载好
2. 拷贝代码
3. 在浏览器（推荐chrome）中按F12打开控制台
4. 在console标签下输入代码(输入前保证输入区域没有其他代码)
5. 回车
6. 再按F12关闭控制台
